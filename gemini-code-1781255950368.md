# Пример: Справочник по сопротивлению материалов

Этот пример показывает, как BOOK-NAV V1 превращает классический справочник в индексируемую базу знаний.

- **Источник:** Справочник по сопротивлению материалов (М.И. Любошиц, Г.М. Ицкович, 1969)
- **Файлы:**
    - `nav.md` — Интеллектуальная карточка.
    - `utility.md` — Профиль применимости.

*Примечание: Оригинальный PDF-файл не включен в репозиторий для соблюдения авторских прав.*
