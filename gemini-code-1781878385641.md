# AAM-V1 Skill Passport: BOOK-NAV System

Этот репозиторий содержит методологию BOOK-NAV V1.6 для систематизации инженерных знаний.

## Суть метода
Метод позволяет трансформировать любой технический источник (учебник, справочник, атлас) в три стандартизированных артефакта:
1. **INDEX** — семантическая карта.
2. **NAV** (MD) — навигатор для человека.
3. **UTILITY** (YAML) — профиль для ИИ-агентов.

## Как развернуть навык в новом чате
1. Скопируй содержимое файла `BOOTSTRAP_PROMPT.txt`.
2. Вставь его в начало нового чата с любой LLM.
3. Система перейдет в режим "Оператор BOOK-NAV".
