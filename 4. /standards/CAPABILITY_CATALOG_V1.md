# Каталог инженерных возможностей (V1)
Список стандартизированных функций для описания источников:
- calculate_[parameter] — расчет характеристик.
- validate_[process] — проверка соответствия нормам.
- optimize_[object] — подбор параметров для максимизации эффективности.
- model_[system] — описание поведения системы.
- diagnose_[failure] — поиск причин отказа.
