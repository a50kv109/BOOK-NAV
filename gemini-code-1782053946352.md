# Правила интероперабельности
1. **Тест на удаление:** Если при удалении метаданных (ANALYTICAL_METADATA) смысл файла теряется — стандарт нарушен.
2. **Языковая нейтральность:** Терминология должна быть понятна любому специалисту в домене, независимо от используемой ИИ-модели.
3. **Запрет статики:** FITNESS и UTILITY не записываются в файлы. Они вычисляются в рантайме по запросу.
4. **Приоритет возможностей:** Описание "что можно сделать" выше описания "что написано".
