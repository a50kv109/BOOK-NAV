# Справочник по сопротивлению материалов (1969)

## Паспорт
- Автор: М.И. Любошиц, Г.М. Ицкович
- Год: 1969
- Язык: RU
- Страниц: 464
- Категория: Инженерная механика

## Ключевые темы
- Растяжение, сжатие, сдвиг
- Изгиб и кручение валов
- Устойчивость сжатых стержней
- Расчет на циклическую прочность

## Что искать в книге
- [стр. 359] Формула Эйлера для устойчивости
- [стр. 175] Напряжения при изгибе
- [стр. 120] Расчет валов на кручение
- [стр. 88] Таблица моментов инерции

## Структура книги
Главы 1-10: Теоретические основы и статические расчеты.
Главы 11-14: Устойчивость, усталость, динамические нагрузки.

## Ценность книги
Классический инженерный справочник с проверенными методиками расчёта, которые актуальны для КБ.
