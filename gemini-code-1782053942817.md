# Стандарт UTILITY-интерфейса V2.0
## Назначение: Обеспечение интероперабельности (Machine-Readable)

- **INPUT:** Формализованный список входных переменных.
- **OUTPUT:** Ожидаемые выходные данные.
- **USE_CASES:** Сценарии применения (инженерные задачи).
- **LIMITATIONS:** Границы применимости алгоритмов/данных.
- **DEPENDENCIES:** Внешние требования.
- **SUPPORTED_TASKS:** Список решаемых задач (на основе CAPABILITIES).
