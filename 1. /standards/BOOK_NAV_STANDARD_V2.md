# Стандарт BOOK-NAV V2.0
## Назначение: Универсальное описание инженерных источников знаний

### Секция 1: Human Layer (Обязательно)
* **Описание:** Общая суть источника.
* **Назначение:** Какую инженерную проблему решает документ.
* **Ключевые темы:** Основные разделы/области знаний.
* **Возможности (Capabilities):** Список того, что можно сделать с помощью данных из источника.
* **Ограничения:** В каких условиях информация неприменима.
* **Входные данные:** Что нужно предоставить (параметры, условия).
* **Выходные данные:** Какой результат будет получен.

### Секция 2: Machine Layer (Обязательно)
* **Capabilities:** Перечень функций в нотации `verb_noun` (например, `calculate_flow`).
* **Dependencies:** Необходимые для работы условия или другие источники.
* **Trace Links:** Ссылки на связанные документы или стандарты.

### Секция 3: Analytical Metadata (Опционально)
* **Методология:** ENGINEERING_CORE_V1.2_PLUS
* **Метрики (X, Y, Z):** Описание топологии и плотности знаний.
* *Примечание:* Данный блок вторичен. Файл должен быть понятен без него.
