# BOOK-NAV

Стандарт интеллектуальной навигации по инженерным книгам.

## Цель проекта
Превратить статические PDF-библиотеки в индексируемые базы знаний. Система автоматически генерирует для каждой книги стандартный файл **NAV.md** — интеллектуальную карточку содержания.

## Стандарт NAV.md
NAV.md — это "карта местности", которая позволяет человеку или ИИ-агенту за 30 секунд понять:
- Что находится в книге (Паспорт, Структура).
- Где искать нужные данные (Формулы, Таблицы, Иллюстрации).
- Какова ценность источника.

## Философия
Книга перестает быть "черным ящиком". С BOOK-NAV вы получаете не просто PDF, а готовый к работе инженерный инструмент.

## Структура
- `server.ts` — движок генерации на базе Gemini.
- `src/` — интерфейс для загрузки и навигации.
- `examples/` — примеры сгенерированных навигаторов.
