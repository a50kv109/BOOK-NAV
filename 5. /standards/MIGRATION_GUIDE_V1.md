# Руководство по миграции
1. Проанализировать существующий документ.
2. Сформировать Human Layer согласно V2.
3. Выделить CAPABILITIES в формате `verb_noun`.
4. Сформировать UTILITY-файл как интерфейс для ИИ.
5. Вынести аналитические метрики в секцию ANALYTICAL_METADATA.
6. Провести тест: удалить метаданные и убедиться, что документ понятен.
